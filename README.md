# Safari Math Game

A safari-themed multiplication game for practicing facts from 2 through 12. Players answer questions, meet a rotating crew of animal guides, and unlock a timed night boss battle.

## Run locally

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Run the build and test suite with:

```bash
npm test
```
